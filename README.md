# Forumazing

This is a personal application to showcase skills with AngularJS and Spring.

Forumazing allows users to communicate and share ideas in a fun and easy to use platform.


## Tech Stack

### Front end
* AngularJS, 
* Twitter Bootstrap

### Back end
* Spring MVC Rest
* Spring Data
* Hibernate
* Spring Security

### Database
* PostgreSQL

### Tools
* Git
* Spring Tool Suite (STS)

### Ops
* Tomcat 7
* OpenShift
